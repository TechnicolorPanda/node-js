# The Odin Project: NodeJS

## Description

This project creates the backend for a simple webpage using NodeJS. It was created as part of [The Odin Project](https://www.theodinproject.com/paths/full-stack-javascript/courses/nodejs/lessons/basic-informational-site)'s Basic Informational Site lesson.

## Objectives & Outcomes

This is the first project in the NodeJS program. The objective is to make an informational site that includes 4 pages.
